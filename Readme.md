# Flip & Frame Companion for iPad (Neon Future Edition)

サイバーパンク・ネオンテイストを取り入れた、iPad向けのスタイリッシュな卓上時計・タイマー＆フォトフレームWebアプリケーションです。画面の左側で時間を確認しながら、右側で好きな写真のスライドショーを表示したり、タイマー機能を利用したりできます。

## 🚀 主な機能
- **ネオンパルス・クロック**: 1分ごとに文字が回転するスタイリッシュなフリップアニメーション。
- **日付＆曜日表示**: 2026年の現在時刻に完全対応。
- **ネオン・フォトフレーム**: 右上のボタンから複数の画像を選択し、30秒間隔で自動フェード切り替え（タップで手動切り替えも可能）。
- **ビルトイン・タイマー**: フォトフレーム部分を下にスワイプすることで、いつでもタイマー画面にアクセス可能（バックグラウンドのズレを抑える高精度設計）。

## 🛠 使い方
1. 本リポジトリの `index.html`（または該当のHTMLファイル）をブラウザで開きます。
2. iPadの「ホーム画面に追加」機能を使って全画面で起動すると、より没入感のあるデザインを楽しめます。
3. 右上の「＋ Upload Neon Photos」から、お気に入りの画像を複数アップロードして使用してください。

## 📱 推奨環境
- iPad OS (Safari / Chrome)
- 画面の自動スリープをオフにして、卓上時計として使用することをおすすめします。
